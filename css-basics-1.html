What is CSS?

CSS

C - Cascading S - Style S - Sheets

CSS is short for Cascading Style Sheets, it is the way to get the preffered look of a website. If HTML is the Skeleton then CSS is the Clothes. It allows you to change colors, size and position of text and tags.

The style sheets define the colour, size and position of text and other HTML tags, while the HTML files define the content and how it is organised. Separating them allows you to change the colour scheme without having to rewrite your entire web site.

Specifying and Using Styles

There are three main ways of including a style sheet for a web page or site:
  1. Setting the style="?" attribute of a tag, called inline styles
    • Inline elements only affect their respective element.
  2. Using the <sytle> tag within the HTML header tag
  3. Using the <link> tag in the <head> to link to an external CSS file

The main difference between the three options is specificity. Specificity determines when a style overrides the other. Inline styles have the highest specificity. This can make them hard to control (you should avoid !important as well for the same reason). Inline styles should only be used as a last resort.

I prefer to use linked stylesheets because:

  • The stylesheet can be cached by browsers for performance
  • Easier to maintain and reuse

How does it work?

The browser scans the HTML document and finds the content and pairs it with the stylesheet using the DOM (Document Object Model).
