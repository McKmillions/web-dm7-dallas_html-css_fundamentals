Element Attributes

The class attribute allows you to target the name specified in your stylesheet and change it's look.

  1. A space between it and the element name (or the previous attribute, if the element already has one or more attributes.)
  2. The attribute name, followed by an equals sign.
  3. An attribute value, with opening and closing quote marks (single `` or double "") wrapped around it.

I prefer to use classes for styling and id's for actions like buttons or forms.
